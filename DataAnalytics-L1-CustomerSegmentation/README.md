# Customer Segmentation Analysis using K-Means

## Oasis Infobyte Internship – Data Analytics

### Task 2: Customer Segmentation Analysis

## Objective

The objective of this project is to segment customers into different groups based on their purchasing behaviour using the K-Means Clustering algorithm. This helps businesses understand customer behaviour and create targeted marketing strategies.

## Dataset

- Dataset Name: Mall Customers Dataset
- Source: Kaggle
- Records: 200
- Features:
  - CustomerID
  - Gender
  - Age
  - Annual Income (k$)
  - Spending Score (1-100)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow

- Loaded and explored the dataset
- Checked missing values
- Performed descriptive statistics
- Selected Age, Annual Income, and Spending Score for clustering
- Standardized data using StandardScaler
- Applied the Elbow Method to determine the optimal number of clusters
- Performed K-Means clustering
- Visualized customer segments
- Profiled each cluster
- Suggested marketing strategies for each customer segment

## Results

- Dataset successfully segmented into **5 customer groups**
- Customer behaviour visualized using scatter plots
- Cluster profiling performed using average feature values
- Marketing recommendations prepared for each customer segment

## Conclusion

Customer segmentation using K-Means clustering provides valuable insights into customer purchasing patterns. Businesses can use these segments to personalize marketing campaigns, improve customer satisfaction, and increase revenue.
